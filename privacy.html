Privacy & Policy

AuroraNova – Weather Monitoring System

1
Information We Use
AuroraNova does not collect or store any personal information. The location entered by the user is used only to fetch real-time weather data and is not saved on the server.
2
Use of Weather Data
Weather information displayed on this website is obtained from trusted third-party services such as the OpenWeatherMap API and is used solely for informational purposes.
3
Cookies
AuroraNova does not use cookies, tracking tools, or analytics services. User interactions are temporary and session-based.
4
Data Security
As no personal data is stored, the risk of data misuse is minimal. API communication is performed using secure connections whenever available.
